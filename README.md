# Support-Ticket Chat UI – Pure HTML & CSS

A two-panel help-desk layout:  
- Left side = searchable ticket list with priority badges  
- Right side = open chat thread + reply box  

Zero JavaScript, zero images, zero dependencies.  
Open `index.html` and the interface is ready.

---

## What you see
- **Tickets panel**  
  – Search bar  
  – Filter buttons: All, Open, Pending, Resolved  
  – Ticket cards: ID, subject, priority (Critical/High/Medium/Low), customer avatar, elapsed time, unread dot  

- **Conversation panel**  
  – Chat header: ticket title, customer name, status dot, flag/link/more buttons  
  – Message bubbles: customer + support team, left/right aligned, time stamps  
  – Reply box: textarea + Send button
